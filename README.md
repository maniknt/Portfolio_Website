# 🚀 MERN Portfolio Website

A modern and fully responsive **MERN Stack Portfolio Website** with a secure Admin Dashboard to manage projects, resume, and contact messages. Built using React, Node.js, Express.js, MongoDB, JWT Authentication, and Cloudinary.

---

## ✨ Features

- 🔐 Secure Admin Login (JWT Authentication)
- 📂 Add, Edit & Delete Projects
- ☁️ Cloudinary Image Upload
- 📄 Resume Management
- 📩 Contact Form with Message Storage
- 📋 Admin Dashboard to View & Delete Messages
- 📱 Fully Responsive Design
- ⚡ REST API Architecture
- 🎨 Clean & Modern User Interface

---

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Portfolio_Website.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

### Server (.env)

```env
MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

CLOUD_NAME=your_cloudinary_name

API_KEY=your_cloudinary_api_key

API_SECRET=your_cloudinary_api_secret
```

### Client (.env)

```env
VITE_API_URL=http://localhost:5000
```
